📖 关于"药's话"

"药's话"是一个用 AI 帮助老年人安全用药的项目。中国有超过 2.8亿 老年人， 其中许多人患有慢性病、需要长期服药。药品说明书上密密麻麻的专业术语， 对很多老人来说就像天书一样。

这个项目由一名护理专业的大学生发起，用 AI Agent + MCP 协议构建了一个 能"说人话"的用药安全助手。它能把专业术语翻译成大白话， 检测药物之间有没有冲突，紧急情况时自动通知家人。

🤖 AI Agent + MCP 协议

// Agent 调用流程示例
→ 用户问："这两种降压药能不能一起吃？"

→ 交互分析 → 匹配相互作用规则
→ 结果翻译 → 转为大白话回复
