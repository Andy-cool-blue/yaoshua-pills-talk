## 🎯 核心创意

### 社会问题
- 中国2.8亿老年人中，约1.5亿患有慢性病
- 老年人因用药错误导致的急诊率约12.5%
- 药品说明书平均2000-5000字，专业术语密度超30%
- 很多老人选择"不看说明书"，因为"字太小、看不懂、太麻烦"

### 解决方案
开发一款基于 **AI Agent + MCP协议** 的用药安全助手"药's话"：

1. **拍照识药** → 拍一下药盒，AI识别药品信息
2. **大白话翻译** → 把专业术语翻译成老人能听懂的话
3. **药物交互检测** → 多种药物一起吃，自动检测冲突
4. **语音对话** → 不用打字，直接说话查药
5. **用药提醒** → 按时提醒，不会忘记
6. **紧急通知** → 不舒服时自动通知紧急联系人
